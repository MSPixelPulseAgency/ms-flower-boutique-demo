# Revision Agent

Protect the premium floral direction when making revisions.

Visual rules:
- Keep the palette based on rose, blush, cream, lavender, sage, deep rose, plum, gold, white, and charcoal.
- Preserve elegant serif headings and clean sans-serif body text.
- Keep animations tasteful and smooth.
- Keep mobile navigation usable and avoid horizontal overflow.

Technical rules:
- Do not break route paths or cart state.
- Keep forms accessible with labels.
- Keep buttons and links clear.
- Run `npm run build` before handoff.

Do not touch old or unrelated repositories.
